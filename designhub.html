STAGE 1

Stage 1 of this project was to re-create an exact replica of their current website that was constructed using wix templating software in order to quickly move off the wix platform and onto another web hosting provider. This was a very quick and easy part of this project. I first assisted the client in setting up their new hosting provider and transfering their existing url. Due to this site being originally being constructed with wix templating software, I was unable re-use any existing code. So I set out to re-create the code from scratch using HTML5, CSS, and Javascript based on the current design template. As I went I made a few simple changes that would improve the responsivity of the website across all browsers and mobile devices.

STAGE 2

Stage 2 of this project was to re-design the website in order to make a more visually appealling webpage, make it more responsive, make it easier for thier current and potential clients use and navigate, and to help increase sales of their design services and bring in new leads. With this part of the project I worked together with the creative director of Designhub as a team to create a new design concept. After recieving the project brief, I began with market research to get a thorough idea of who their users are, what they want, and what they would like to see when choosing a broadcast design branding company. I also conducted research examining current design trends in order to be at the forefront of industry. I then met together with the creative director to discuss the project, the results of my research and brainstorm ideas. We then jottted down some ideas and compiled together images of design inspiration. I started with creating wireframe mockups of the design using Sketch2 and used photoshop for editing various assets used in the site. I also recieved design ideas from the creative director via photoshop and incorporated the ideas into the sketch wireframe.

As the layout took shape I filled out more details of the design to create a high-fidelity prototype. Throughout this project I coached the creative director on the technical aspecs of the design from the view point of a web developer. I did have a design challenge. We liked the modern look of a transparent navigation menu, but it was often unreadable depending on what was showing underneith it. If I put a background color on the navigation bar it would ruin the look of the clean modern design. So to solve this quandry, I came up with the idea of using a transparent linear gradient in the background starting from slightly transparent to fully transparent. Thus making the background dark enough to view the links without sacrificing the asthetic design.

STAGE 3

Stage 3 of this project was to take the new design layout and develop the website using HTML, CSS, and Javascript, make it responsive across all browsers and devices and upload it to their new web hosting server. I developed this site using video studio code and used two node modules, Rellax and Yet Another Lazy Loader, for parallax and lazy load features. These helped to quickly streamline the development process. This project will soon expand to stage 4 in the near future to incorporate backend development features to help manage its large number of media content and to improve security.
